Apa yang dimaksud database

Database adalah kumpulan data yang terorganisasi

Apa singkatana dan pengertian dari RDBMS?

Rasional Database Management System : software atau aplikasi yang menggunakan rasional database model sebagai dasarnya

Sebutkan 5 cara apa saja yang dipakai dalam penyimpanan data pada database model!

  • Flat Model : Menyimpan data secara bebas tanpa aturan dan tata cara tulis tertentu
  • Hierarchical Model : Menyimpan data terorganisasi dari atas kebawah
  • Network Model : Merupakan pengembangan dari Hierarchical Model
  • Rational Model : Menyimpan data kedalam bentuk tabel yang saling terhubung
  • Object Oriented Model : Penyimpanan data dengan cara pendekatan dalam object
